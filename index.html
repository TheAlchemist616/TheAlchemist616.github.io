Multi-Stream Race Overlay

Free browser-based Twitch multi-stream overlays for race restreams. Add as an OBS Browser Source — no installs needed.

Quick Start

  1. Open the Interactive Editor in your browser
  2. Press E to enter edit mode
  3. Pick a preset layout, type in channel names, adjust panels
  4. Click Export as URL to copy the configured link
  5. In OBS, add a Browser Source → paste the URL → set to 1920×1080

Interactive Editor

Full drag-and-drop overlay editor. Press E to toggle edit mode. You get a sidebar with all controls and a visible 1920×1080 canvas grid.

Layout Controls

Preset buttons — quickly set up 2, 3, 4, 6, 8, 10, 12, 16, or 24 stream grids
+ Add Stream — add a panel manually to the canvas
Drag a panel to move it anywhere on the canvas
Resize handles — drag corners/edges to resize (hold Shift to lock 16:9)
Channel names — editable in the sidebar stream list or detail panel
Delete key or button to remove a panel

Crop Mode (Zoom/Pan)

Select a panel, then click Enter Crop Mode in the sidebar. The stream plays live while you adjust what part of it is visible:

Scroll wheel — zoom in/out (1x to 4x)
Drag on the stream — pan to frame the area you want
Escape or button to exit crop mode
In regular edit mode: Shift+drag to pan, scroll to zoom

Saving & Sharing

Auto-saves to browser localStorage on every change
Export as URL — shareable link with layout encoded in the URL hash
Copy Config JSON — raw config for backup
Open an exported URL on any machine to load that exact layout

Static Overlay Files

Simple fixed-grid alternatives if you don't need the full editor. Set channel names via URL parameter:

thealchemist616.github.io/6-streams.html?channels=user1,user2,user3,user4,user5,user6

OBS Setup

  1. Add a Browser Source
  2. Set dimensions to 1920×1080
  3. Paste your overlay URL
  4. The background is transparent — layer your frame/border overlay on top

Bandwidth

Each stream uses roughly 3–6 Mbps of your download bandwidth. The streams load in your browser (or OBS), not on a server. More panels = more bandwidth.
